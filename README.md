## Description

Nest framework TypeScript starter repository with typeorm and graphql.

## Installation

```bash
$ npm install
$ cp development.example.env development.env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Homepage http://localhost:3000

Graphql http://localhost:3000/api/graphql

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
